# Documentation Map

This directory contains the main AI-OS documentation.

## Main areas

- `ai-os/` - master operating manual
- `methodology/` - state machine, definition of done, quickstart, maturity model
- `diagrams/` - Mermaid diagram catalog
- `loops/` - scenario-specific engineering loops
- `verifiers/` - completion and evidence gates
- `architecture/` - system architecture
- `model-routing/` - local-first model and MCP policy
- `memory/` - project memory model
- `security/` - secure SDLC and supply chain guidance
- `ci-cd/` - workflows, releases, wiki sync
- `evaluations/` - eval strategy
- `local-runner/` - runner concept
