# Continuous Improvement Prompt

```text
Execute AI-OS continuous improvement loop.

Goal:
[IMPROVEMENT GOAL]

Inspect the repository first.
Fan out analysis to documentation, governance, security, CI/CD, reader experience, and prompt specialists.
Create a prioritized plan.
Ask for human approval before implementation.
After approval, implement in small reversible steps.
Verify structure, links, diagrams, workflows, and documentation.
Update memory, roadmap, and wiki navigation.
Repeat until no high-value improvement remains in the approved scope.
```
