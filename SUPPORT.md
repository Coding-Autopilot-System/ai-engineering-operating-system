# Support

AI-OS is a public methodology and documentation framework.

## Best support channels

- Use GitHub issues for documentation gaps, loop improvements, prompt improvements, and examples.
- Use pull requests for concrete improvements.
- Use private security reporting for vulnerabilities or unsafe autonomous-agent behavior.

## What to include

- Goal
- Current behavior
- Desired behavior
- Suggested loop or verifier
- Acceptance criteria
- Verification evidence if available
